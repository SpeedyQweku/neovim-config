## Install

#### Deps

- fd
- neovim
- rg
- python-pynvim

```bash
[ -f ~/.config/nvim/ ] && mv ~/.config/nvim ~/.config/nvim-backup  
git clone https://github.com/punixcorn/neovim-config ~/.config/nvim
```

#### Run neovim to complete

```bash
nvim
```
